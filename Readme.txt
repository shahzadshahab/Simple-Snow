A lightweight, elegant snowfall effect for WordPress. Optimized for speed & performance with light & dark mode.

== Description ==

**Simple Snow** adds a beautiful, elegant snowfall animation to your WordPress site during the winter or holiday season ‚Äî without slowing it down.

This plugin is intentionally minimal:  
- **No performance impact** (GPU-friendly canvas animation).  
- **No clutter** ‚Äî only one simple setting.  
- **No libraries or dependencies**, not even jQuery.  
- **No shortcodes, no widgets, no blocks**, fully automatic.  

You can enable a light or dark color profile so the snow remains visible on all theme backgrounds.

== Features ==

- Lightweight snowfall effect using HTML5 Canvas

- Random natural wind movement

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/simple-snow/` or install via the Plugin Directory.  
2. Activate the plugin through **Plugins ‚Üí Installed Plugins**.  
3. Go to **Settings ‚Üí Simple Snow** to select **Light** or **Dark** background mode.  
4. The snowfall effect will instantly appear on your site.

== Frequently Asked Questions ==

= Will this slow down my site? =  
No. The animation is canvas-powered and optimized to use minimal resources. Most browsers use GPU acceleration for smooth performance.

= Can I adjust the number of snowflakes? =  
Not yet ‚Äî the plugin is intentionally minimal, but more options may be added based on feedback.

= Does it work on mobile? =  
Yes. The canvas automatically scales and adjusts density.

= Does it affect SEO? =  
No. It does not modify any HTML or content structure ‚Äî only renders on the visual layer.
